# VLAN & Inter-VLAN Routing Lab 

This lab demonstrates network segmentation using VLANs, 802.1Q trunking, and Inter-VLAN routing (Router-on-a-Stick).

## Objectives
- Create multiple VLANs
- Assign access ports
- Configure 802.1Q trunks
- Enable Inter-VLAN routing using a router sub-interface
- Verify connectivity between VLANs

## Technologies Used
- VLANs (10, 20, 30)
- Trunking (802.1Q)
- Router-on-a-Stick
- Cisco IOS
